# Chapter 14: Building C-Level Agents

> *Executive Summary Agents and Strategic Insights*

---

## 14.1 What Makes a C-Level Agent

A C-Level agent provides:
- Executive summaries
- Cross-system insights
- Risk flagging
- Strategic dashboards
- Natural language interaction with data

Target users: CEO, CFO, executive teams.

---

## 14.2 Cross-System Orchestration

C-Level agents typically pull from multiple sources:
- S/4HANA (financials, operations)
- SuccessFactors (HR data)
- Ariba (procurement)
- Analytics Cloud (BI reports)


Map each system to a destination and create skills for each.

---

## 14.3 Managing AI Unit Consumption

**Warning**: C-Level agents with grounding + multi-step reasoning consume AI credits fast!

Tips:
- Start with simpler models in dev
- Monitor usage in BTP Cockpit
- Set usage alerts
- Optimize prompts for efficiency

---

## 14.4 Practical Tips for Executive Dashboards

1. **Keep responses concise** — Executives want summaries
2. **Ground on real data** — Not generic responses
3. **Handle confidentiality** — Restrict who can ask what
4. **Test with real scenarios** — "What's our Q3 outlook?"
5. **Iterate with stakeholders** — Get executive feedback early
